**Pig_Latin_Generator

**Theory

This code generates PigLatin from English phrases on the command line.

**Why

This code was written in 30 minutes for clear a live coding round at a major naval communication firm. The PigLatin rules were created on the spot. In Hindi, we call this "Cha ki Bhasha" (or the language of Cha). Chidren can learn at speak that rapidly to avoid detection. The PigLatin code  also removes punctuation.

**How to test

This code has been tested at JDoodle (https://www.jdoodle.com/online-java-compiler/)

**Sample input

* test test -> esttay esttay esttay 
* oink oink -> oinkay oinkay 
* a man, a plan, a canal  : panama
